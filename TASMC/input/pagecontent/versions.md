## Versioning Policy

- Versions follow **Semantic Versioning**: `MAJOR.MINOR.PATCH`
- Only **one** version is marked as `current`
- Older versions remain published for traceability but may be marked `Retired`
- Breaking changes increment **MAJOR**
- Backward-compatible changes increment **MINOR**
- Clarifications and fixes increment **PATCH**

---

## FHIR Compatibility

All published versions of this guide currently target:

- **FHIR R4 (4.0.1)**

Future FHIR versions will be published under a new major version.
